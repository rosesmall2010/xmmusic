## 📝 使用说明

### 主要功能入口

1. **音乐扫描**: 设置 → 媒体库 → 添加目录
2. **播放控制**: 底部播放器
3. **搜索**: 顶部搜索框
4. **播放列表**: 侧边栏 → 我的歌单
5. **收藏**: 音乐列表中的心形图标
6. **ID3 修复**: 设置 → ID3修复
7. **均衡器**: 设置 → 均衡器
8. **元数据编辑**: 音乐列表右键 → 编辑元数据
9. **智能推荐**: 侧边栏 → 智能推荐
10. **播放统计**: 侧边栏 → 播放统计
