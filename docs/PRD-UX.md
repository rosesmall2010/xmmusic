## 🎨 用户体验设计

### 设计原则

1. **简洁现代**：清晰的视觉层次，减少视觉噪音
2. **年轻化**：符合年轻用户审美的设计风格
3. **流畅交互**：流畅的动画和过渡效果
4. **一致性**：统一的视觉语言和交互模式

### 界面布局

**整体布局**：头部 + 中部 + 底部

- **头部**：应用名称、搜索框、主题切换、语言切换、窗口控制
- **中部**：侧边栏（菜单） + 主内容区
- **底部**：播放控制区

### 主题系统

- **浅色主题**：白色背景，深色文字
- **深色主题**：深色背景，浅色文字
- **主题切换**：即时生效，无需重启

### 交互设计

- **动画效果**：平滑过渡（0.2-0.3s）
- **操作反馈**：按钮悬停、点击效果
- **拖拽交互**：播放列表排序、添加到播放列表

## 📐 UI 布局详细说明

### 头部区域

**布局**:
```
[Logo: xmmusic] [Search Input] [Theme] [Lang] [Min] [Max] [Close]
```

**设计要点**:
- Logo 使用品牌色 #ff4757
- 搜索框：圆角 20px，宽度 250px
- 按钮：30x30px，圆形，悬停效果
- 关闭按钮：悬停时红色背景

### 中部区域

**左侧侧边栏**:
- 🎵 本地音乐
- 🕒 最近播放
- 📋 我的歌单（可展开）
- ❤️ 我的收藏（可展开）
- ⚙️ 设置

**右侧主内容区**:
- 音乐列表视图
- 播放列表视图
- 设置视图
- 流派分组视图

### 底部区域

**播放控制区**:
- 左侧：封面 + 歌曲信息
- 中间：播放控制按钮 + 进度条
- 右侧：歌词、播放模式、播放列表、音量、音质

## 🎨 视觉设计

### 颜色方案

**浅色主题**:
- 主背景：#ffffff
- 次背景：#f5f5f5
- 文字：#333333
- 强调色：#ff4757

**深色主题**:
- 主背景：#1a1a1a
- 次背景：#2d2d2d
- 文字：#ffffff
- 强调色：#ff4757

### 字体

- **基础字体**: 系统字体栈
- **基础字号**: 14px
- **标题字号**: 18px, 16px

### 间距

- **基础单位**: 8px
- **常用间距**: 8px, 16px, 24px, 32px

## 🖱️ 交互设计

### 动画效果

- **页面切换**: 淡入淡出（0.2s）
- **菜单展开**: 高度动画（0.3s）
- **按钮悬停**: 背景色过渡（0.2s）
- **列表项悬停**: 背景色过渡（0.15s）

### 操作反馈

- **点击**: 按钮按下效果
- **悬停**: 背景色变化
- **成功**: 绿色提示（0.5s）
- **错误**: 红色提示（2s）
- **警告**: 黄色提示（2s）

### 拖拽交互

- **播放列表排序**: 拖拽排序
- **添加到播放列表**: 拖拽添加
- **列顺序调整**: 拖拽调整

## 📱 响应式设计

### 窗口尺寸适配

**小窗口 (< 1000px)**:
- Sidebar 可折叠
- 列表列数减少
- Footer 按钮简化

**大窗口 (> 1600px)**:
- 内容区域最大宽度限制
- 居中显示
