# 统一风格指南

## 目标
统一主界面、正在播放界面和 Mini 窗口的视觉风格，确保用户体验的一致性。

## 设计原则

### 1. 统一使用 CSS 变量系统
所有界面必须使用 CSS 变量，支持深色/浅色主题自动切换。

**核心 CSS 变量：**
- `--bg-color`: 主背景色
- `--bg-secondary`: 次背景色
- `--text-color`: 主文字颜色
- `--text-secondary`: 次要文字颜色
- `--text-tertiary`: 三级文字颜色
- `--border-color`: 边框颜色
- `--hover-bg`: 悬停背景
- `--active-bg`: 激活背景
- `--color-primary`: 主色调

### 2. 背景系统统一
- **主界面**：使用 `--bg-color`（主题背景色）
- **正在播放界面**：动态背景（封面颜色渐变）+ 主题适配的终点色
  - 深色主题：封面色 → `#0a0a0a`
  - 浅色主题：封面色 → `#f5f5f5`
- **Mini 窗口**：使用 `--bg-color`（主题背景色）

### 3. 文字颜色统一
- 主文字：`--text-color`
- 次要文字：`--text-secondary`
- 三级文字：`--text-tertiary`
- 禁用文字：`--text-disabled`

### 4. 按钮和控件统一
- 主按钮：使用 `--color-primary`，白色文字
- 次要按钮：透明背景，`--text-color` 文字
- 悬停背景：`--hover-bg`
- 激活背景：`--active-bg`
- 边框颜色：`--border-color`
