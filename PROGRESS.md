## 📝 决策记录

### UI 重构 (仿QQ音乐)
- **日期**: 2025-11-22
- **决策**: 采用仿QQ音乐的现代化UI风格
- **理由**: 提供更熟悉、更现代的用户体验，提升视觉质感

### 图标库迁移
- **日期**: 2025-11-24
- **决策**: 从 Emoji 迁移到 Lucide Icons
- **理由**: 提升界面统一性和专业度，解决跨平台显示差异

### 技术选型
- **日期**: 2025-11 初
- **决策**: 使用 Electron 39 + Vue 3 + TypeScript
- **理由**: 成熟稳定，社区活跃，TypeScript 提供类型安全
