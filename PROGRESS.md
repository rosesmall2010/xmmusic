## 📝 决策记录

### 技术选型
- **日期**: 2025-11 初
- **决策**: 使用 Electron 39 + Vue 3 + TypeScript
- **理由**: 成熟稳定，社区活跃，TypeScript 提供类型安全

### 安全架构
- **日期**: 2025-11-19
- **决策**: 启用 sandbox 和 webSecurity
- **理由**: 提高应用安全性，符合 Electron 最佳实践

### SQLite 实现
- **日期**: 2025-11 初
- **决策**: 使用 @vscode/sqlite3
- **理由**: 与 Electron 39 兼容性最好
