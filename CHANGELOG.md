# 变更记录

所有重要的项目变更都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.1] - 2025-11-20

### 新增
- 开发环境和生产环境数据库分离（开发环境使用 `xmmusic-dev.db`）
- 播放队列状态图标，实时显示歌曲是否在播放队列中
- 关于页面添加详细的技术栈说明

### 修复
- 修复均衡器功能不工作的问题
  - 将 audio 元素添加到 DOM，使 Web Audio API 能够连接
  - 改进均衡器初始化和缓存更新逻辑
- 修复均衡器页面切换时崩溃的问题（添加数据验证）
- 修复歌单功能缓存更新问题
- 静默处理数据库关闭时的 SQLITE_BUSY 错误（不影响数据完整性）

### 改进
- 统一术语：将"播放列表"改为"歌单"（Playlist），"播放队列"（Queue）概念更清晰
- 移除播放队列操作成功时的弹窗提示，图标状态变化已足够提示用户
- 优化用户体验，减少不必要的提示弹窗

### 技术栈更新
- 更新关于页面显示的技术栈版本信息
  - Electron 39.2.1
  - Vue 3.5.24
  - TypeScript 5.9.3
  - Vite 7.2.2
  - 新增 Pinia 和 Howler.js 展示

## [1.0.0] - 2024-XX-XX

### 新增
- 初始版本发布
- 本地音乐扫描和管理
- 音乐播放功能
- 歌单管理
- 收藏功能
- 播放队列
- 均衡器（音效设置）
- 高级搜索
- 元数据编辑
- 相似歌曲推荐
- 导出功能（Excel、文件）
- SQLite 数据库存储
- 跨平台支持（macOS、Windows、Linux）
